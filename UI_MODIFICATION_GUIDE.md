## 二、问题清单与修改建议

### 2.1 注册页面 ❌ 问题严重（优先级 P0）

#### 当前问题
- 输入框和文字标签位置混乱，重叠在一起
- 文字在输入框左侧外面，没有居中对齐
- "用户名" 标签跑到输入框外面了
- 输入框之间没有间距

```
当前布局（错误）：
用户名                                        邮箱地址
[          ] [          ]     ← 两个输入框重叠，且标签不在框内

密码              确认密码
[          ] [          ]     ← 同样的问题
```

#### 修改建议

| 元素 | 当前值 | 建议值 |
|------|--------|--------|
| 用户名输入框 y 值 | -1543 | -1500 |
| 用户名标签位置 | x:3437, y:-1597 (错误) | x:3460, y:-1512 (框内垂直居中) |
| 邮箱输入框 y 值 | -1543 (重叠) | -1420 |
| 邮箱标签位置 | x:3437, y:-1543 (重叠) | x:3460, y:-1432 |
| 密码输入框 y 值 | -1467 | -1340 |
| 确认密码输入框 y 值 | -1443 | -1260 |
| 输入框高度 | 48px | 44px |
| 输入框圆角 | 0px | 8px |
| 输入框间距 | 混乱 | 80px（等距）|

#### 目标效果
```
┌─────────────────────────────────┐
│                                 │
│         创建账号                 │  ← 居中，28px，加粗
│                                 │
│  ┌─────────────────────────┐   │
│  │  用户名                  │   │  ← 浅灰文字提示
│  └─────────────────────────┘   │
│                                 │
│  ┌─────────────────────────┐   │
│  │  邮箱地址                │   │
│  └─────────────────────────┘   │
│                                 │
│  ┌─────────────────────────┐   │
│  │  密码                   │   │
│  └─────────────────────────┘   │
│                                 │
│  ┌─────────────────────────┐   │
│  │  确认密码               │   │
│  └─────────────────────────┘   │
│                                 │
│     [     注册     ]           │  ← 红色按钮，全宽
│                                 │
│      已有账号？登录             │  ← 红色文字链接
└─────────────────────────────────┘
```

### 2.4 个人资料页面 ❌ 内容单薄（优先级 P1）

- 只有头部信息，内容太单薄
- 缺少 Tab 切换
- 缺少帖子展示

#### 详细设计建议

```
┌────────────────────────────────────────┐
│ ┌────────────────────────────────────┐ │
│ │                                    │ │
│ │         封面图片区域               │ │  ← 灰色或图片，200px
│ │                                    │ │
│ └────────────────────────────────────┘ │
│     ┌────────┐                          │
│     │ 头像   │  用户昵称                │  ← 头像120px，重叠在封面上

│                                                  │
│  [帖子] [相册] [收藏]  ← 红色下划线选中         │
│ ───────────────────────────────────────────
│ ┌────────┐ ┌────────┐ ┌────────┐
│ │        │ │        │ │        │  ← 帖子网格，3列
│ │  图片  │ │  图片  │ │  图片  │
│ │        │ │        │ │        │
```

| 元素 | 数值 |
